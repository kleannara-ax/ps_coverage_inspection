DB 연동 v5.0.0

이미지 준비

점보롤 샘플 이미지를 업로드하여 분석을 시작하세요.

이미지 선택

* 최대 900MB 이하의 이미지 파일을 지원하며, 조명 반사가 적은 이미지를 사용할수록 결과가 안정적입니다.

검사 메타데이터

MES/공정 연동 필수 값을 입력하세요. * 표시는 필수 항목입니다.

검사항목그룹코드, 자재코드(MATNR), 플랜트(WERKS), 점보롤 LOT 번호는 MES API 연동 시 자동으로 수신됩니다.

수신된 값은 검사 결과 저장 시 DB에 기록됩니다.

측정일시(MSRM_DATE)와 검사 수행 시각(inspected_at)은 이미지 업로드/분석 시 자동 기록됩니다.

지표 주석

  • 총 지분: 자동 검출된 지분에서 제외하지 않은 개수와 수동으로 추가한 지분을 합산한 값입니다.
  • 밀도: 지름이 약 3px 이상 7px 이하로 추정되는 지분의 개수입니다.
  • 총 커버리지: 전체 이미지 픽셀 대비 지분 픽셀 비율(ppm)입니다.
  • 크기 균일도: 지분 픽셀 면적의 평균과 표준편차로 계산한 0~100점 지표입니다.
  • 분포 균등도: 이미지를 4분면으로 나누었을 때의 지분 분포 균형도입니다.
